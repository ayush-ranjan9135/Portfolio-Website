Project Title: "Girl Safety - SOS Application"

Description: I developed a Girl Safety - SOS App to provide emergency assistance with a single tap. The app allows users to send SOS alerts via SMS to emergency contacts, ensuring quick help in critical situations. It features real-time location sharing, contact management, and Firebase integration for secure data storage.

Project Title: "Tic-Tac-Toe Game"

Description: I developed a Tic-Tac-Toe Game with an interactive and user-friendly interface. Players can compete against each other with smooth gameplay and real-time win detection. Built using Java and XML, the app offers a fun and engaging experience.".

Project Title: "To-Do List App"

This is a simple and user-friendly To-Do List web app where users can add, edit, delete, and mark tasks as completed. It also saves tasks in the browser using localStorage, so the list stays even after refreshing the page. A progress bar shows task completion, and a fun confetti animation appears when all tasks are completed!

Project Title:"Calculator App"

This is a basic and easy-to-use calculator web app made with HTML, CSS, and JavaScript. It supports simple arithmetic operations like addition, subtraction, multiplication, and division. The design is clean and responsive, and it includes buttons like AC (All Clear), DEL (Delete), = (Equals), and number/operator inputs for real-time calculations.
